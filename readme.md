##Lego Mobile
Lego Mobile是基于zepto和seajs的轻量级移动组件库，提供了一系列webapp的UI解决方案。支持UC，QQ，Opera，Safari，Chrome等国内主流手机浏览器。


## Quick Start for building

### Prepare the environment

1. node, npm
2. [grunt v0.4](http://gruntjs.com/getting-started) - `npm install grunt-cli -g`
3. grunt-init `npm install grunt-cli -g`
4. ruby, gem, [bundler](http://gembundler.com/)
5. sass `gem install sass`

## Demo

1. `grunt demo`
